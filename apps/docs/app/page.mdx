# React Unstyled Swipe Button

A lightweight, unstyled, and fully accessible swipe-to-action button for React.

- **Zero Dependencies:** Built from scratch with no external packages.
- **Fully Customizable:** Bring your own styles. Use CSS, CSS-in-JS, or Tailwind CSS.
- **Accessible:** Built with ARIA attributes and keyboard navigation in mind (future step).
- **Modern & Smooth:** Fluid animations using React state and CSS. Works great on all devices.

---

This documentation will guide you through installing the component, understanding its props, and creating your own custom variants.
